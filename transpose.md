# Transpose the lines of a file

USAGE `cdx transpose [options...] [file]`

||||
|---|---|---|
| -h | --head| Print ` CDX` before other data |
| -l | --lines=Number | Maximum lines to examine |
| -u | --unique | remove duplicate lines. |

for example:
```
   1  2  3
   4  5  6
```
becomes
```
   1  4
   2  5
   3  6
```
